# Number of Trailing Zeros of N!

## Instructions

Write a program that will calculate the number of trailing zeros in a factorial of a given number.

## Examples

zeros(6) = 1
6! = 720 --> 1 trailing zero

zeros(12) = 2
12! = 479001600 --> 2 trailing zeros
